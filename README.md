# 🪑 Shree Ram Furniture Website

A modern, responsive business website developed for **Shree Ram Furniture**, a furniture business based in Pune, Maharashtra.

This project was built for a real client to provide an online presence, showcase services, and allow customers to submit furniture enquiries through a custom contact form.

---

## 🌟 Features

- Responsive design for desktop and mobile
- Modern furniture showcase
- Hero image slider
- Service sections
- About section
- Contact information
- Custom enquiry form
- W3Forms integration
- Cloudflare Pages hosting
- Smooth scrolling navigation
- Form validation
- Clean and lightweight design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- W3Forms
- Cloudflare Pages
- GitHub

---

## 📩 Contact Form

The enquiry form is integrated with **W3Forms**, allowing visitors to submit enquiries without requiring a backend server.

Submitted enquiries include:

- First Name
- Last Name
- Email
- Phone Number
- Space Type
- Furniture Requirement
- Customer Message
